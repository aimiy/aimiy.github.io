# 常用像素处理公式
## 灰色rgb公式：
arg = (r + g + b) / 3

r = arg

g = arg

b = arg

[一些优化的视觉上不同灰度处理公式](http://www.tannerhelland.com/3643/grayscale-image-algorithm-vb6/)等有时间翻译出来

## 反色(手机相机中的负片)rgb公式
r = 255 - r
g = 255 - g
b = 255 - b
