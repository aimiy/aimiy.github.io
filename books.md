P7(技术专家)

 精通各种Web前端技术（HTML/CSS/Javascript等)，熟练跨浏览器、跨终端的开发；

 精通React、AngularJS、Vue等MVVM框架至少一种，并有在大型项目中使用的成功经验；

 熟悉 Node.js Web 应用开发，有大型 Node.js 项目的开发经验；

 熟悉 Web 安全相关知识，并能使用相关技术防范安全漏洞；

对前端技术有持续的热情，个性乐观开朗，逻辑性强，善于和各种背景的人合作；
